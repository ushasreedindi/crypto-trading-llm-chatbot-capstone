# crypto-trading-llm-chatbot-capstone
Capstone project analyzing cryptocurrency trading strategies and evaluating LLM chatbots with RAG.
## Project Overview
This project focuses on solving two fintech challenges:

1. Developing a cryptocurrency trading strategy using technical indicators.
2. Evaluating Large Language Models (LLMs) to build a customer-support chatbot with Retrieval-Augmented Generation (RAG).

## Trading Strategy
The project implemented a trading strategy using technical indicators:

- RSI (Relative Strength Index)
- SMA (Simple Moving Average)
- EMA (Exponential Moving Average)
- ATR (Average True Range)

### Key Results
- Bitcoin Return: **16.77%**
- Ethereum Return: **15.93%**
- Sharpe Ratio: **1.24**

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- yFinance
- Technical Analysis Indicators (RSI, SMA, EMA)
- Large Language Models (Llama, Mistral, Gemma)
- Retrieval Augmented Generation (RAG)

## Chatbot Development

LLMs evaluated:

- Llama
- Mistral
- Gemma
- XGen

Configurations tested:
- Standard LLM
- LLM + RAG
- LLM + RAG with tailored responses

### Evaluation Metrics
- BLEU
- ROUGE
- BERTScore
- Cosine Similarity

## Key Findings
Llama with RAG showed the best performance across semantic and lexical evaluation metrics.
